# frontend-lzt-connect

Base frontend em Next.js pensada para servir como camada visual de um sistema que ja possui backend proprio.

## O que esta incluido

- Next.js 14 com App Router
- TypeScript
- Tailwind CSS
- Estrutura simples e componentizada
- Layout responsivo como base inicial

## O que foi removido

- Banco de dados e migrations
- Autenticacao embutida
- Testes automatizados
- Storybook
- Monitoramento e integracoes extras do boilerplate original
- Referencias ao repositorio original

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
- `npm run check-types`

## Estrutura principal

- `src/app`: paginas da aplicacao
- `src/components`: componentes reutilizaveis
- `src/templates`: estrutura base de layout
- `src/styles`: estilos globais
- `src/utils`: configuracoes do projeto
